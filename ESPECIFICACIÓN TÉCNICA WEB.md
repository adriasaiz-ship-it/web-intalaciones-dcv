# **DOCUMENTO 2**

# **ESPECIFICACIÓN TÉCNICA WEB**

## **PROYECTO: INSTALACIONES DCV 2024**

Este documento define los requisitos técnicos y funcionales para el desarrollo de la web en la infraestructura Antigravity.

---

# **1\. INFRAESTRUCTURA**

La web se desarrollará en Antigravity utilizando una arquitectura optimizada para rendimiento.

Alojamiento:

CDN para carga rápida.

Certificado SSL obligatorio.

Backups automáticos.

---

# **2\. RENDIMIENTO**

La web debe cumplir con los estándares de Google Core Web Vitals.

LCP  
 menos de 2.5 segundos

FID  
 menos de 100 ms

CLS  
 menos de 0.1

---

# **3\. ESTRUCTURA DE URL**

Las URLs deben ser limpias y SEO friendly.

Ejemplo:

/electricistas-tarragona  
 /fontaneros-tarragona  
 /aire-acondicionado-tarragona  
 /telecomunicaciones-tarragona

No se permitirán parámetros extraños.

---

# **4\. SEO TÉCNICO**

Debe implementarse:

sitemap.xml  
 robots.txt

---

# **5\. MARCADO DE DATOS**

Implementación de Schema LocalBusiness mediante JSON-LD.

Debe incluir:

nombre empresa  
 teléfono  
 dirección  
 servicios

---

# **6\. FORMULARIOS**

Campos obligatorios:

Nombre  
 Teléfono  
 Email  
 Servicio

Tras el envío aparecerá un popup:

"Solicitud enviada con éxito. Nos pondremos en contacto contigo lo antes posible."

---

# **7\. NOTIFICACIÓN INTERNA**

Los datos enviados se enviarán únicamente a los administradores por correo electrónico para que contacten personalmente.

---

# **8\. PROTECCIÓN ANTI SPAM**

Protección mediante:

reCAPTCHA v3  
 honeypot

---

# **9\. TRACKING**

Integración con:

Google Analytics 4

Eventos a medir:

envío de formularios  
 clic en teléfono  
 clic en WhatsApp

---

# **10\. GOOGLE MAPS**

Integración de mapa interactivo en la página de contacto.

---

# **11\. SEGURIDAD**

La web debe incluir:

SSL  
 protección anti spam  
 actualizaciones automáticas  
 copias de seguridad

---

# **12\. CUMPLIMIENTO RGPD**

Debe incluir:

banner de cookies  
 aviso legal  
 política de privacidad  
 política de cookies

Checkbox obligatorio en formularios.

---

# **13\. DISEÑO RESPONSIVE**

La web debe diseñarse con enfoque Mobile First.

Más del 70% del tráfico se espera desde móvil.

---

# **14\. FOOTER**

El pie de página debe incluir:

dirección  
 teléfono  
 email  
 servicios  
 enlaces legales

---

# **15\. PÁGINA 404**

Debe incluir:

mensaje de error  
 botón volver al inicio  
 botón ver servicios

---

# **CONCLUSIÓN**

Esta especificación técnica garantiza que la web sea:

rápida  
 segura  
 optimizada para SEO  
 orientada a captar clientes

y preparada para el crecimiento futuro de **INSTALACIONES DCV 2024**.
